The results include three separate leaderboards:

* **`leaderboard_tournament_adjusted.csv`** - for players with tournament history
* **`leaderboard_rank_adjusted.csv`** - for players without tournament history
* **`leaderboard_adjusted.csv`** - the combined leaderboard of all players (prioritizing tournament-adjusted rankings when a player appears in both)
